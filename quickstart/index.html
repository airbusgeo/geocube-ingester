Getting started

This quickstart requires docker-compose.

If you don't have a Geocube Server running, you can start one, with docker-compose, following these instructions. Here the quickstart is configured for a geocube server running on localhost:8080, without apikey.

  • Clone https://github.com/airbusgeo/geocube-ingester.git.
  • Start geocube-ingester service with docker-compose, following these instructions.

The ingester is running and waiting for your instructions !

Then, you can do the tutorial to learn how to use the ingester.
